# Bodymovin 插件中英文对照汉化指南

## 1.项目概述
使用 Cursor AI 协助完成 bodymovin 插件的中英文对照汉化工作，格式要求为：`汉化文本（英文原文）`

## 2.文件结构
```
项目根目录/
├── 待汉化源文件/          # 原始英文文件
├── 待汉化源文件//build/  # 已汉化参考文件（仅中文）
```

## 3.汉化格式要求
- **标准格式**：`汉化文本（英文原文）`
- **示例**：`获取播放器（get the player）`
- **注意**：保持英文原文在括号内，便于开发者理解
